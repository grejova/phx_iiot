# Repository Scope

This repository is a public technical evidence repository for PHX IIoT.

It is not an open-source release of the core deployment engine, Ignition gateway scripts, internal orchestration logic, or production implementation code.

The purpose of this repository is to make the system concept, runtime behavior, validation direction, and safe public artifacts reviewable without exposing implementation-sensitive industrial automation code.

Public materials may include:

- architecture and governance notes
- demo topology source definitions
- controlled test evidence
- screenshots and diagrams
- field pilot preparation notes
- non-sensitive deployment logs or summaries

Core implementation details remain private because the project involves industrial runtime provisioning patterns, Ignition-specific deployment logic, and operational automation concepts that may be sensitive in real plant environments.
